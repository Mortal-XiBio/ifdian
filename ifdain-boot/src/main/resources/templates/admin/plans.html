如何获取 Cookie？

方法一：控制台复制（推荐）

  1. 在浏览器中打开 ifdian.net 并登录
  2. 按 F12 打开开发者工具，切换到 Console 面板
  3. 粘贴以下代码并回车，即可复制 Cookie：
// 复制此代码到 ifdian.net 的控制台执行
(function(){
  var cookies = document.cookie.split(';').map(function(c){
    var parts = c.trim().split('=');
    return {name: parts[0], value: parts.slice(1).join('=')};
  }).filter(function(c){ return c.name; });
  // 用 JSON 格式输出（推荐，包含更多信息）
  var json = JSON.stringify(cookies, null, 2);
  console.log('=== Cookie JSON (可直接粘贴到系统设置) ===');
  console.log(json);
  console.log('=== 或使用 document.cookie 格式 ===');
  console.log(document.cookie);
  copy(json);
  console.log('✅ 已复制 JSON 格式到剪贴板！');
})();

方法二：手动导出

  1. Chrome/Edge: F12 → Application → Cookies → ifdian.net → 逐个复制名称和值
  2. 粘贴到系统设置时使用 JSON 数组格式：[{"name":"auth_token","value":"xxx"},{"name":"user_id","value":"yyy"}]
  3. 或者直接粘贴 document.cookie 的输出结果（name1=val1; name2=val2 格式）
