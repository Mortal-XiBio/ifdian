查询方案详情
外部支付请求 API

供外部程序调用的 REST API，用于创建支付请求、查询支付状态和手动重试回调。

鉴权方式

所有接口需携带 API Key，方式二选一：

  • 请求头: X-Api-Key: your-api-key
  • 查询参数: ?api_key=your-api-key

API Key 在 系统设置 → 外部 API 配置 中管理。

端点列表
方法路径说明
POST /api/external/payment-request 创建支付请求，返回支付链接和签名密钥
GET /api/external/payment-request/{requestId} 查询支付请求的状态和详情
POST /api/external/payment-request/{requestId}/retry-callback 手动重试支付回调（仅已支付状态可用）
curl 示例

创建支付请求：

curl -X POST https://your-domain.com/api/external/payment-request \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: your-api-key" \
  -d '{
    "customOrderId": "ORDER-2024001",
    "planId": "a45353328af911eb973052540025c377",
    "callbackUrl": "https://your-app.com/payment-callback",
    "remark": "VIP 月度订阅"
  }'

回调签名验证代码示例

Ifdain 回调时会携带 X-Ifdain-Signature 请求头，使用 signatureSecret 对请求体做 HMAC-SHA256 签名。请验证签名以确保请求来源可信。

  • Java
  • Python
  • Node.js
import javax.crypto.Mac;
import javax.crypto.spec.SecretKeySpec;
import java.util.Base64;

public boolean verify(String body, String signature, String secret) {
    Mac mac = Mac.getInstance("HmacSHA256");
    mac.init(new SecretKeySpec(
        Base64.getDecoder().decode(secret), "HmacSHA256"));
    String expected = Base64.getEncoder()
        .encodeToString(mac.doFinal(body.getBytes()));
    return expected.equals(signature);
}
import hmac, hashlib, base64

def verify(body: str, signature: str, secret: str) -> bool:
    key = base64.b64decode(secret)
    expected = base64.b64encode(
        hmac.new(key, body.encode(), hashlib.sha256).digest()
    ).decode()
    return hmac.compare_digest(expected, signature)
const crypto = require('crypto');

function verify(body, signature, secret) {
    const expected = crypto
        .createHmac('sha256', Buffer.from(secret, 'base64'))
        .update(body)
        .digest('base64');
    return crypto.timingSafeEqual(
        Buffer.from(expected), Buffer.from(signature));
}
